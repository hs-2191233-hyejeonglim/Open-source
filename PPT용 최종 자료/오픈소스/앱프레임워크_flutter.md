# 앱 제작 프레임워크 : Flutter

- 오픈소스 설명
  - 구글에서 제작한 dart 언어의 하이브리드 앱 제작용 오픈소스 프레임워크
  - 우리가 설계한 애플리케이션의 프론트엔드를 제작하는데에 사용
- 데이터 입출력
  - 입력 및 출력 : 프론트엔드 프레임 워크이기 때문에 사용자의 디바이스에 출력되고 입력되는 거의 모든 데이터가 통과함 
  - DFD에서는 사용자의 디바이스와 동일하게 취급
